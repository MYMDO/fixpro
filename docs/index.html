📦 Installation

1

Flash the Firmware

Connect your Pico in BOOT mode and copy the firmware file:

cp FiXPro.uf2 /media/your-user/RPI-RP2/
2

Install Python CLI (optional)

For command-line control:

pip install pyserial
python -m fixpro detect
3

Start Flashing

Use this web interface or the CLI to read/write your chips.
